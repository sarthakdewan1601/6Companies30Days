# Question Statement-

Given two library versions of an executable: for example, "10.1.1.3" and "10.1.1.9" or "10" and "10.1". Find out which one is more recent? Strings can be empty also.

### Sample Input 1
```
10.1.1.3
10.1.1.9
```
### Sample Output 1
```
10.1.1.9
```
<br><br>

### Sample Input 2
```
10
10.1
```
### Sample Output 2
```
10.1
```
